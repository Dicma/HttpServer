HttpResponse Class

[This is preliminary documentation and is subject to change.]

Class handles everything related to server response to client. It is related to HttpRequest and HttpServer. It is created by HttpServer from HttpRequest object. All data writen by write method goes to the requesting stream. Allows to add cookies and headers to the server response. It allso automatically adds session cookie if there is session.
